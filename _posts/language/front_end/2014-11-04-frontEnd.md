[html,css,javascript基础知识](http://www.w3school.com.cn/)   
	
	html： 内容  
	javascript：事件方法  
	css:样式(纯粹的个人理解)。      
下边是我对三者之间的关系。
   
*	html 文档对象*  
	*	全局属性  ---> css
	*	事件属性  ---> javascript    
		*	windows事件  
		*	form事件			
		*	keyboard事件	  	
		*	mouse事件  
		*	media事件  

&nbsp;&nbsp;&nbsp;&nbsp;通过*`html`*标签生成文档对象，然后通过*`css`*来控制它的属性：宽高位置颜色等等，通过*`javascript`*来控制它在某事件发生时触发的动作。
